## Machine-learning Project
- 2022-1 광운대학교 기계학습 텀프로젝트

## Project Subject
- 의료데이터를 통한 응급환자 예측
- 응급실 환자가 12시간 이내에 사망할지, 중환자실로 이송될지 예측
- 기대효과 : 응급실 환자가 얼마나 위험한지 의사의 객관적인 판단 도움, 오진단 위험 ▼, 한정된 자원의 효율적 분배
## Data
- MIMIC-IV-ED database

## Model

- Gradient Boosting Classifier
- XGBoost
- LightGBM
